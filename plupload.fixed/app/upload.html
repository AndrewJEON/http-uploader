Файлозагрузчик

You browser doesn't have Flash or Silverlight support.

Сервис загрузки файлов на сервер, тестовая версия.

  • Работает (проверено) в конфигурациях  Windows XP SP3 + Mozilla Firefox 3.6.13 + (Silverlight Plug-In 3.0.5, Shockwave Flash Plug-In 10.1.102). В других комбинациях не проверял, но работать должен с любым браузером, оснащенным плагином Флеш или Сильверлайт.
  • С Сильверлайт работает лучше (работает с файлами более гигабайта), с Флеш работает хуже (может не принять файл более 500 мегабайт). Память утекает с Сильверлайтом тоже заметно медленнее.
  • Рекомендую не грузить файлы более 300 (допустимо) - 500 (максимум) мегабайт за раз. Если файл большего размера, разбейте его на куски, например с помощью WinRAR.
  • Докачка в этой версии не поддерживается, поэтому, если сбойнул коннект, текущий файл придется перезаливать с самого начала. Это еще одна причина дробить большие файлы на меньшие части.
  • При загрузке, на вашей системе расходуется оперативная память, до 2-х мегабайт оперативки на мегабайт файла.
  • Поскольку файлы загружаются кусочками, в принципе загрузить можно файл любого размера, но учитывая приведенные ограничения, файл более гигабайта залить вряд ли получится. Если надо больше, разбейте файл на куски.
